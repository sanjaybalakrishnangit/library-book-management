# Library Book Management System

## Problem Statement
To design and implement a Library Book Management system using MongoDB with schema validation, CRUD operations, and error handling.

## Technologies Used
- MongoDB
- Express.js
- React.js
- Node.js

## Features
- Book schema with validation
- Insert, Read, Update, Delete operations
- Conditional delete (copies = 0)
- Error handling
- Frontend display using React

## Project Structure
- backend/ - Node.js & Express backend
- frontend/ - React frontend

## How to Run
1. Start MongoDB
2. Run backend:
node server.js
3. Run frontend:
npm run dev
